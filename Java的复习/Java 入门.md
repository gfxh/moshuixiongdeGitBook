

# 什么是计算机语言？
用于人与计算机之间通讯的语言，是人与计算机之间传递信息的媒介。计算机系统的最大特征是通过一种语言传达指令给机器。

为了使电子计算机进行各种工作，需要有一套用以编写计算机程序的数字、字符和语法规则，由这些字符和语法规则组成计算机各种指令（或各种语句）。
计算机语言可以分为三大类：机器语言、汇编语言和高级语言，一般计算机世界里把“机器语言和汇编语言”叫做低级语言。

Java 是一种高级语言，你可能听过的其他高级语言还有 Python、C、C++ 等。实际上计算机只能运行低级语言，所以高级语言编写的程序必须先被翻译成低级语言才能运行。

目前世界上主要的编程语言翻译方式有

* 解释(interpreting)
* 编译 (compiling)

完成解释任务的程序叫解释器，解释器会按行（一行一行的读取）对程序进行翻译，然后执行命令。

完成编译任务的程序叫做编译器，它会在高级语言执行之前一次性对该程序进行完整的编译，所以程序的运行是在编译过程之后。我们一般把这种高级语言称为**源代码**
(source code,我们经常说的 coding 就是写源代码),编译器生成的程序称为可执行程序（executable）。
# Java 是什么？
简单说，Java 是计算机高级语言的一种，一门非常流行、强大且耐用的编程语言。

*它有几个核心特点让你一听就懂：*
1. **“一次编写，到处运行”​​**

Java 比较强大，它既可以被编译也可以被解释，不同的是 Java 编译过程并不生成机器语言，而是生成字节码。
字节码和机器语言一样，但是它具备高级语言的可移植性，因此在一台机器上编译 Java，在另外一台机器上也可以被解释运行，这就是 Java 语言能够在诞生的时候就瞬间超越了其他编程语言的核心原因:
这是 Java 最牛的地方。你用 Java 写的程序，编译后可以在 Windows、Mac、Linux 上直接运行，无需修改。这背后的功臣是 ​JVM（Java 虚拟机）​，它像一个万能翻译官。

2. **面向对象**

它用“对象”的思维来组织代码，让开发大型项目更容易管理和维护。比如，把“汽车”看作一个对象，它有颜色、品牌等属性，有启动、加速等方法。

3. **自动管理内存​**

Java 自带“垃圾回收”机制，会自动清理不用的内存。程序员不用像用 C/C++ 那样操心内存释放，减少了错误和负担。

4. **应用广泛​**

    Java 几乎无处不在：

    * 安卓 App 开发​：大部分手机 App 的后台都是用 Java 写的。

    * 企业级后台系统​：银行、电商、互联网公司的大型服务器后端。

    * ​大数据​：Hadoop、Spark 等大数据技术都用 Java。

    * Web 开发​：很多网站的后台逻辑也是 Java 处理的。

​**一句话总结**：Java 是一门功能全面、稳定性高、就业市场需求巨大的编程语言，尤其擅长开发复杂的大型企业级应用。​
